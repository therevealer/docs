# zkPass

zkPass is a **zero-knowledge oracle protocol** that enables users to prove data from Web2 and Web3 sources without exposing the underlying data.

## What zkPass Solves

- Prove private data without revealing it
- Eliminate trusted data custodians
- Enable composable, on-chain verification
